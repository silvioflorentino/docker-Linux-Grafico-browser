Existem várias formas de rodar um Linux com interface gráfica dentro de um container Docker. A maneira mais simples hoje é usar uma imagem que já vem com ambiente gráfico acessível pelo navegador (VNC + noVNC).

Pré-requisitos

  • Docker instalado
  • Pelo menos 4 GB de RAM livres
  • Navegador web

Verifique se o Docker está funcionando:

docker --version
docker run hello-world

Opção 1 (Mais fácil): Ubuntu gráfico acessível pelo navegador

1. Baixe a imagem

docker pull dorowu/ubuntu-desktop-lxde-vnc

2. Execute o container

docker run -d \
  --name ubuntu-grafico \
  -p 6080:80 \
  -p 5900:5900 \
  dorowu/ubuntu-desktop-lxde-vnc

Parâmetros:

  • 6080 → acesso pelo navegador
  • 5900 → acesso via cliente VNC
  • -d → executa em segundo plano

3. Acesse a interface gráfica

Abra no navegador:

http://localhost:6080

Você verá uma área de trabalho Ubuntu/LXDE rodando dentro do Docker.

4. Entrar no terminal do container

docker exec -it ubuntu-grafico bash

5. Instalar programas

Dentro do container:

apt update
apt install -y firefox

Depois execute:

firefox

A janela aparecerá na área gráfica do navegador.

Opção 2: Ubuntu gráfico moderno (XFCE)

Baixar

docker pull accetto/ubuntu-vnc-xfce

Executar

docker run -d \
  --name ubuntu-xfce \
  -p 6081:6080 \
  accetto/ubuntu-vnc-xfce

Acessar:

http://localhost:6081

Opção 3: Linux gráfico usando Docker Compose

Crie um arquivo docker-compose.yml:

services:
  ubuntu:
    image: dorowu/ubuntu-desktop-lxde-vnc
    container_name: ubuntu-grafico
    ports:
      - "6080:80"
      - "5900:5900"

Suba o ambiente:

docker compose up -d

Pare:

docker compose down

Salvar seus arquivos

Monte uma pasta do host:

docker run -d \
  --name ubuntu-grafico \
  -p 6080:80 \
  -v ~/meus-arquivos:/home/ubuntu/shared \
  dorowu/ubuntu-desktop-lxde-vnc

Tudo que estiver em:

~/meus-arquivos

aparecerá dentro do container.

Verificar containers

Listar:

docker ps

Parar:

docker stop ubuntu-grafico

Iniciar novamente:

docker start ubuntu-grafico

Remover:

docker rm -f ubuntu-grafico

Exemplo completo

docker pull dorowu/ubuntu-desktop-lxde-vnc

docker run -d \
  --name ubuntu-grafico \
  -p 6080:80 \
  -v ~/dados:/home/ubuntu/shared \
  dorowu/ubuntu-desktop-lxde-vnc

Depois abra:

http://localhost:6080

e você terá um Ubuntu com interface gráfica rodando dentro do Docker.
